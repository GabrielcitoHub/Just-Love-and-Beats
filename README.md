# HOW TO INSTALL

Windows: (without git)
Download the code as a zip, unzip, drag unzipped folder to love.exe, done!

Windows: (with git)
Clone the reposistory, drag folder to love.exe, done!

Any other platform: (without git)
Download the code as a zip, unzip, select all files on folder, zip,
rename the ".zip" in the filename to ".love", drag that to love.exe, done!

Any other platform: (with git)
Clone the reposistory, drag folder to love.exe, done!

Android: (with git and termux) (NOT SUPPORTED YET)
Run "termux-setup-storage" (if you haven't already) run ls, choose a folder
to clone into, Clone the reposistory (if it says you don't have it just install
it with the command it prompts), open the LOVE loader or open it with an explorer,
done!
